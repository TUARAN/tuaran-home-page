## 一、基本信息

- 成立时间：2021 年
- 总部：美国旧金山
- 创始团队：Dario Amodei、Daniela Amodei 等前 OpenAI 核心成员
- 融资历史：累计融资金额超百亿美元，主要投资方包括 Google、Amazon
- 主营业务：通用大模型（Claude 系列） + Claude API + Claude Code

## 二、产品与技术

- **Claude 系列模型**：Opus / Sonnet / Haiku 三档，覆盖高性能到高性价比场景
- **Claude Code**：面向开发者的终端 Agent，支持工程化代码任务
- **Claude API（原 Anthropic API）**：标准化推理接口，支持 Tool Use、Prompt Caching、扩展思考等
- 技术路线：长期投入 RLHF / Constitutional AI / 可解释性研究

## 三、商业化

- 收入模型：API 计费 + Claude.ai 订阅 + 企业服务
- 关键客户：覆盖科技、金融、咨询、政府等
- 与 AWS Bedrock、Google Vertex AI 双云分发

## 四、近期动作

- 2026-Q1：Claude 4.7 Opus 发布
- 2025：Claude Code 形成稳定产品形态，配套 Skills、Hooks、Plugins 体系

## 五、值得关注的点

- Agent 工程化能力是当前最稳定的差异化优势
- "扩展思考 + Prompt Caching" 是 API 侧最值得跟进的两个方向
- Claude Code 的 Skill / Plugin 生态有可能演化为新的开发者基础设施
