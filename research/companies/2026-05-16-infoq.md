## 一、基本信息

- **品牌**：InfoQ（中国区）；旗下还包括极客时间、QCon、ArchSummit、AICon 等产品 / 活动品牌
- **国际母品牌**：InfoQ.com（C4Media Inc. 旗下，加拿大）
- **中国运营主体 / 公司全称**：北京极客邦科技有限公司
- **InfoQ 中国上线**：2007-03-28
- **极客邦科技正式成立**：2014 年（以收购 InfoQ 大中华区业务为关键节点）
- **总部**：北京
- **创始人 / CEO**：**霍泰稳**（Floyd 共同把 InfoQ 引入中国，后独立组建极客邦）
- **公司定位**：世界级 IT 知识服务平台（自称）
- **业务集群**：媒体（InfoQ） + 教育（极客时间） + 社交（EGO） + 出版 + 会议 + 培训 + 咨询

> **关系厘清**：很多人混淆"InfoQ vs 极客邦"。准确说法是 —— **InfoQ 是品牌、极客邦是公司**：
> - InfoQ.cn 是极客邦旗下技术媒体板块
> - 极客邦科技是承载方，旗下还有更核心的"极客时间"知识付费业务

## 二、产品与技术

### 2.1 产品矩阵（一句话定位）

| 产品 | 性质 | 一句话定位 |
|---|---|---|
| **InfoQ.cn** | 技术媒体 | 国内权威技术新闻 / 深度文章 / 大会内容源头 |
| **极客时间 App / time.geekbang.org** | C 端知识付费 | 程序员的"得到"，专栏 / 视频课 / 训练营 |
| **极客时间企业版（b.geekbang.org）** | B 端 SaaS | 给企业 IT 团队的统一学习平台 |
| **QCon / ArchSummit / GMTC / AICon** | 线下大会矩阵 | 国内最具影响力的 IT 技术大会系列 |
| **EGO** | CTO / 技术高管社群 | 闭门职业社交 |
| **InfoQ 写作平台** | UGC 内容 | 让外部作者直接发稿到 InfoQ |
| **图书出版** | 实体内容 | 与机械工业 / 电子工业出版社合作出书 |

### 2.2 内容形态

- **InfoQ 文章**：编辑 + 译稿 + 邀稿 + 写作平台 UGC 多源混合
- **极客时间专栏**：付费订阅型长内容（典型 199-299 元 / 专栏）
- **训练营 / 集训营**：高客单价（数千-上万）的 cohort-based 培训
- **大会内容**：线下大会 → 线上视频 → 文章稿件，形成内容飞轮

### 2.3 技术能力

- 自研内容平台、订阅 / 学习系统、企业版 SaaS
- 直播能力（live.geekbang.org）
- 早期 App + 持续 Web 端迭代，**技术工程能力是知识付费厂商里偏强的**

## 三、商业化

### 3.1 收入结构（公开口径，2023 年前后）

| 收入板块 | 占比 | 说明 |
|---|---|---|
| **B 端营销服务 + 线下大会** | **~70%** | 大会赞助、企业品牌服务、定制内容 |
| **C 端知识付费** | **~30%** | 极客时间订阅、专栏 / 视频 / 训练营 |

> 注：**B 端是真正的现金流主轴**，C 端是品牌与流量入口。这跟很多人对"极客时间是主业"的直觉相反。

### 3.2 客户与用户

- **C 端注册用户**：> **160 万**（极客时间口径）
- **B 端付费企业**：~ **2000 家**（含百度等头部互联网公司）
- **大会客户**：阿里、腾讯、字节、华为、美团等几乎所有头部互联网公司

### 3.3 盈利状况

- **公司一直处于盈利状态**（公开自述）
- 是少数"内容 + 知识付费"赛道里能稳定赚钱的企业

## 四、人才与组织

### 4.1 团队

- **核心团队**：以霍泰稳（CEO）为代表的内容 + 媒体 + 教育复合背景
- **技术 + 内容 + 销售**多线条：技术媒体编辑、专栏作者运营、企业销售、SaaS 工程师并存
- 公开渠道未披露员工总数 —— 推断在 **300-500 人量级**（参考 B 端 2000 家客户 + C 端 160 万用户 + 大会运营所需人力）

### 4.2 招聘与薪酬（推断 + 公开口径）

- 北京 IT 媒体 / 知识付费行业薪酬中位水平
- **编辑 / 内容运营**：本科 ¥10K-15K，资深 ¥20K-30K
- **企业销售 / 大客户**：¥15K-30K + 提成（提成空间是该行业的特点）
- **研发**：与一线互联网公司有差距，但**工作强度普遍小于大厂**
- **课程作者 / 讲师**：合作分成模式，头部讲师收入可观

### 4.3 组织文化

- **内容驱动**：员工普遍懂技术、懂行业，对开发者群体有深度同理心
- **2025 年目标 IPO**：意味着组织正在向规范化 / 财务化运营转型

## 五、产业地位

### 5.1 行业定位

- **国内技术媒体第一梯队**：与 CSDN、51CTO、掘金、思否并列五大平台
- **技术大会赛道国内绝对头部**：QCon / ArchSummit 是国内 IT 技术大会的事实标杆
- **知识付费（IT 垂类）赛道头部**：极客时间是 IT 知识付费的事实第一品牌

### 5.2 与同行对比

| 平台 | 主战场 | 与 InfoQ/极客邦的差异 |
|---|---|---|
| **CSDN** | 博客 + 下载 + 资讯 | 流量更大、SEO 更强；商业化更弱、内容质量两极分化 |
| **51CTO** | 学习平台 + 企业培训 | 业务模型更接近，B 端更老牌，但**没有 InfoQ 的内容品牌力** |
| **掘金（字节系）** | 前端 + 创作者社区 | 流量增长更快、调性更"年轻"；缺大会与企业版闭环 |
| **思否（SegmentFault）** | 问答 + 社区 | 体量较小、垂直问答属性 |
| **得到（罗辑思维）** | 通识知识付费 | 跨品类，IT 垂类深度与 InfoQ 差距明显 |

### 5.3 综合卡位

- **"国内 IT 知识服务的事实首选 B 端供应商"** ≈ 唯一的对标定位
- 在"内容 / 大会 / 培训 / 出版 / 社群"五位一体的完整度上**国内无对手**

## 六、估值与产值

### 6.1 公开融资历史

| 时间 | 轮次 | 金额 | 投资方 |
|---|---|---|---|
| 早期 | 天使 | 未披露 | — |
| 2018 | A 轮 | **¥2000 万** | 银虎资本、慧科资本 |
| 近期 | B 轮 | **¥7000 万** | 阳光保险 |

### 6.2 产值估算

- 公开渠道**未披露**最新营收，但可基于以下信号推断：
  - 2000 家 B 端客户 × 平均客单价 ¥5-20 万 ≈ **¥1-4 亿元 / 年 B 端基本盘**
  - 大会业务（QCon / ArchSummit / GMTC 等数十场 / 年）单场赞助 + 门票 ¥数百万-千万
  - C 端付费用户保守估算 30-50 万付费用户 × 客单价 ¥200-500 ≈ **¥0.6-2.5 亿元**
- **估算年营收区间**：**¥2-6 亿元**（行业惯例推断，非披露）
- 行业惯例 PS 3-5x → 估算估值 **¥6-30 亿元**（仅作框架参考）

### 6.3 IPO 进展

- 创始人公开表示 **2025 年有望上市**
- 公开渠道暂无明确递表时间 / 板块（A 股 / 港股 / 美股均有可能）
- **关键问号**：知识付费 + IT 媒体 + B 端 SaaS 多业务混合模型，A 股审核口径友好度待观察

## 七、近期动作

- **2026-04-16~18**：**QCon 全球软件开发大会·北京站（2026）** 收官，主题"**以 AI 之力，重塑软件工程新生态**" —— 大模型 / Agent / 软件工程 AI 化是 2026 年 QCon 的核心叙事
- **2026 年 2 月**：极客邦 / InfoQ 报道智谱"全球大模型第一股"上市首日涨超 13%，强化"AI 第一媒体"定位
- **2026 年初**：与火山引擎数智平台达成合作，聚焦用户精细化运营场景 —— 这是极客邦少数公开的"自身产品 + 字节生态"绑定动作
- **2025 持续动作**：极客时间持续上线 AI 大模型 / 智能体 / 工程化重磅专栏
- **2024 起**：持续投入 SaaS 系统建设（创始人 2023 年表态）
- **持续**：QCon / ArchSummit / AICon 系列大会矩阵；InfoQ 写作平台引入更多开发者作者
- ⚠️ **公开渠道无 2026 年 IPO 实质性进展披露**：与 2023 年"2025 年有望上市"的口径相比，节奏可能延后

## 八、值得关注的点

### 8.1 业务结构层面

- **"内容 / 教育 / 大会 / 社群" 四位一体**是其无可复制的护城河，单看某一项都不算行业最大，但**组合起来形成了无人能复刻的闭环**
- B 端 70% 的收入意味着**经济周期 / 大厂预算波动会直接传导**到极客邦的业绩 —— 这是它的最大风险点
- C 端 160 万用户对 IT 垂类已属高水位，但**增长曲线趋平**，大模型时代的"内容形态"是否要彻底重塑是关键

### 8.2 大模型时代的最大变量

- **AI 内容生产能力让传统"编辑 + 译稿"模式被冲击**：InfoQ 早期靠译稿建立的国际视野优势，正在被 ChatGPT / Claude 直接抹平
- **极客时间专栏的"长视频 + 文字"形态**也面临挑战：用户更愿意问 AI 而不是订阅一门 199 元专栏
- **应对方向**：从"卖内容"转向"卖知识图谱 + Agent + 实战项目陪跑"。能否完成这个转身决定了它是否会被时代抛下

### 8.3 IPO 前的关键不确定性

- 知识付费赛道公开 IPO 案例少（一起教育、网易有道等都是相关但不重合的标的）
- B 端 + C 端 + 出版 + 大会**业务模型复杂**，估值锚定难
- 上市更可能是"为团队 / 老股东退出"，而非"补血扩张"——这意味着上市后增长动能值得严密观察

### 8.4 个人视角

- **InfoQ + 极客邦是国内 IT 内容圈"理想主义气质最浓"的厂商**：长期主义、品牌洁癖、用户口碑都好
- 它的最大价值是**"开发者群体的精神社区感"**，单个产品很难承载这种价值；这种感受很难量化但极难复刻
- 对外部观察者：
  - **如果做内容 / 创作者** → 投稿 InfoQ 写作平台、做极客时间课讲师，是高质量曝光的稀缺通路
  - **如果做企业 IT** → QCon / ArchSummit / 极客时间企业版是构建技术雷达 + 团队学习的"国内最优解"
  - **如果做投资判断** → IPO 故事性足够，但商业模型复杂度 + AI 冲击 + 增长曲线趋平三个变量需谨慎
- 个人结论：**短期看 IPO 落地节奏，长期看大模型时代的内容形态转身**
